# Markdown Preview Studio

실시간 Markdown Preview 웹앱입니다.

## Features
- Live Preview (Editor + Preview)
- GitHub Flavored Markdown (GFM)
- Code Highlight
- Dark Mode
- Auto Save (LocalStorage)
- Export HTML / PDF

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
